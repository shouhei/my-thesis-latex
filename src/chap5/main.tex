\begin{itemize}
  \item[・] 授業進行に無理がなかったか
  \item[・] 授業難易度が適切であったか
  \item[・] 受講者が授業内容を正しく理解しているのか
\end{itemize}

\begin{itemize}

  \item[・] 実験実施後アンケート
\end{itemize}
